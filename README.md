# Number Guessing Game 

This is a simple Number Guessing Game built using HTML, CSS, and JavaScript.

##  Features
- Random number generate hota hai
- User number guess karta hai
- Correct / Wrong message show hota hai
- Simple and beginner-friendly project

## Tech Stack
- HTML
- CSS
- JavaScript

##  How to Play
1. Enter a number
2. Click the submit button
3. Try to guess the correct number
