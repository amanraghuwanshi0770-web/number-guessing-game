# Number Guessing Game

Ye ek simple **Number Guessing Game** hai jo JavaScript ki madad se banaya gaya hai.  
Is game me computer ek random number generate karta hai aur user ko us number ko guess karna hota hai.

---

## Features
- Random number generate hota hai
- User input ke through guess kar sakte ho
- Sahi guess par winning message
- Galat guess par hint (chhota ya bada number)
- Simple aur beginner-friendly project

---

## Technologies Used
- HTML
- CSS
- JavaScript
